В этой папке планируется разместить клиентские приложения (от одного и больше) для визуальной работы с функционалом системы. В частности, это может быть веб-сайт либо приложение под Android, iOS, Windows Phone и т.п. Десктопное приложение для планшетов также может попасть в этот список, при наличии сил, времени и желания.
